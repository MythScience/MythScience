# 神话科学 Myth Science

以科学的方法、严谨的态度，构建中国神话传说的结构化数字工程。

[MythScience.com](https://mythscience.com) · [Hello@MythScience.com](mailto:Hello@MythScience.com)

---

## 这是什么

**神话科学（Myth Science）** 是一个数字人文平台。

中国神话传说散落在大量古籍文献中——同一角色在不同朝代、不同作者笔下可以是完全不同的面貌，同一事件在不同版本中互相矛盾。研究者、创作者、开发者想要利用这些素材，往往得从零开始翻书，翻完还不确定翻对了没有。

MythScience 做的就是把这件事系统化：以结构清晰、来源可溯、逻辑自洽的方式，将中国神话传说中的角色、世界观、神兽异兽、神器法宝、仙草灵植、重要地点、重大事件等整理为开放的数字化档案。每一句断言都经得起追问——出处在哪？哪个版本？有没有争议？

## 能做什么

基于这套结构化数据，可以做的事情包括但不限于：

- **学术研究**：跨体系角色对比、文献版本差异追溯、神话谱系梳理
- **数据可视化**：关系图谱、时间线、地理分布等交互式探索
- **创作参考**：为小说、游戏、影视等创作提供可溯源的原始文献依据
- **AI 应用**：基于档案数据的角色扮演、知识问答、内容生成等
