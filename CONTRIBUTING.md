# 贡献指南

欢迎为 MythScience 做出贡献。

## 贡献方式

### 报告问题

发现数据错误、文档问题或有新角色建议？请[提交 Issue](https://github.com/MythScience/MythScience/issues/new)。

Issue 请包含：
- 涉及的文件路径或角色名称
- 问题描述（数据错误、缺失、建议等）
- 如有文献出处，请附上

### 提交新角色档案

1. 先开 Issue 说明计划创建的角色及所属体系，避免重复工作
2. 按 `Archive/Characters/{体系中文名}/{角色中文名}/` 路径创建目录
3. 档案文件使用 Markdown 格式，UTF-8 编码
4. 每条事实性断言须标注文献出处——精确到回目/章节
5. 提交 Pull Request，标题格式：`[新建] {体系}/{角色名称}`

### 修正既有档案

1. 提交 PR，描述中说明修改的文件、内容和依据
2. 如有新增出处标注，注明文献和回目

## Pull Request 规范

- PR 标题清晰描述改动内容
- 关联对应的 Issue（如有）
- 所有档案数据以原文为准，不确定的标注 `[待考]` 或 `[存疑]`

## 行为准则

参与本项目即表示同意遵守[贡献者公约](CODE_OF_CONDUCT.md)。

## 许可

贡献即视为同意将内容以 [CC BY-NC 4.0](LICENSE) 许可发布。
