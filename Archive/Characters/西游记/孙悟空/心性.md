# 孙悟空——心性

## 当前状态

### 终态信念

| 信念 | 含义 | 出处 |
|------|------|------|
| 自由是一切价值的根基 | 从求长生到闹天宫到反复求褪紧箍——自由是他贯穿一生的最高价值。成佛后紧箍自去，自由终获圆满 | [全期验证：P1忧生死→P2求长生→P3闹天宫→P4反复求褪紧箍→P5紧箍自去，第一百回] |
| 强者不必低于任何人 | 从"皇帝轮流做，明年到我家"到称佛后第一句话就直问紧箍——成佛了不代表他会在权威面前放低姿态 | [D-对话, 第七回（对如来说"强者为尊该让我"）；第一百回（成佛后直问唐僧紧箍）] |
| 恩义必报 | 菩提赶他走时他说"念师父厚恩未报"；被唐僧逐出时叩头谢恩；对观音始终恭敬有加 | [D-对话, 第二回、第二十七回、全书多处] |
| 真相必须大白 | 被冤枉时极度痛苦——三打白骨精被逐时反复争辩，真假美猴王时走遍天宫地府只为一个"辨明邪正" | [D-叙述, 第二十七回（三打白骨精）、第五十七至五十八回（真假美猴王）] |

### 终态性格

| 特征 | 性质（正面/负面/中性） | 表现 | 出处 |
|------|---------------------|------|------|
| 勇敢 | 正面 | 面对任何妖王从不退缩，十万天兵面前毫无惧色 | [全期验证] |
| 机智 | 正面 | 变小钻风探狮驼洞、变牛魔王骗芭蕉扇、变仙丹入妖魔腹中——善用计谋而非蛮力 | [全期验证] |
| 忠诚 | 正面 | 被逐后仍嘱咐沙僧"就说老孙是他大徒弟"；被猪八戒请回后毫无怨言继续保师 | [D-对话, 第二十七回、第三十一回] |
| 幽默 | 正面 | 全期用"呆子""夯货"调侃八戒，在最紧张的战斗中仍有闲情戏谑 | [全期验证] |
| 好胜 | 中性 | 不能容忍被轻视或被误解。被叫"弼马温"即暴怒；被冤枉时争辩到底 | [全期验证] |
| 直率 | 中性 | 想什么说什么，不藏话。对如来直说"如来不该慈悯他"；成佛后直问紧箍 | [全期验证] |

| 傲慢 | 负面 | P3时自称"强者为尊该让我"；P4中对八戒、土地神常带居高临下的口气 | [D-对话, 第七回、多回] |

### 终态驱动力

| 驱动类型 | 内容 | 依据 |
|------|------|------|
| 欲望（Want） | 不受约束的自由——从求长生（不受死亡约束）到闹天宫（不受天庭约束）到反复求褪紧箍（不受咒语约束） | [D-对话, 第一、四、七、十四、二十七、五十八、一百回] |
| 需求（Need） | 被认可和尊重——弼马温之辱、齐天大圣有官无禄之虚、三打白骨精被冤之痛，都指向他深层需要的是：在自由的基础上，被他人真正承认和尊重 | [推测，基于：①弼马温得知官小后的暴怒（第四回）②竖"齐天大圣"旗本身就是对"你不配上座"的抗议③被逐时的"鸟尽弓藏"之叹（第二十七回）④成佛后第一件事确认紧箍消失（第一百回）——他不是在乎那个佛号，是在乎那个箍所代表的"你还没被当成人"] |
| 核心恐惧 | 被遗忘和被抛弃——三打白骨精被逐时在海面上哭；真假美猴王时无处可去、四顾茫然；"鸟尽弓藏，狗烹兔死"的自比 | [D-叙述, 第二十七回（在海面上哭）、第五十七回（四顾无去处）] |

### 终态矛盾

成佛后的孙悟空存在一个微妙的内在矛盾——斗战胜佛"是佛门果位，但"斗"和"战"这两个字恰恰是他原始本性的标签。如来不是压服了他的本性，而是**承认了它**：你爱打，好，你的打是"炼魔降怪有功"，封你为"斗战胜佛"。这种"成佛不灭本性"的安排，让他不必在"做自己"和"成正果"之间二选一。

## 成长轨迹

### P1：石猴

**此阶段信念**：
- 世界是好玩的——"春采百花为饮食，夏寻诸果作生涯"
- 死亡是可怕的——"一旦身亡，可不枉生世界之中？" [D-对话, 第一回]

**此阶段性格**：
| 特征 | 性质 | 表现 | 出处 |
|------|------|------|------|
| 天真 | 中性 | 无忧无虑，不知生死为何物 | [D-叙述, 第一回] |
| 勇敢 | 正面 | 第一个跳入水帘洞瀑布 | [D-叙述, 第一回] |
| 聪明 | 正面 | 发现水帘洞后立刻利用承诺要求众猴拜王 | [D-叙述, 第一回] |
| 好奇 | 正面 | 对未知世界充满探索欲 | [D-叙述, 第一回（出海）] |

**此阶段驱动力**：欲望——长生不老。需求——超越"石猴"这个原始身份（所以出海前"将石字隐了"称"美猴王"）。

**关键变化说明**：P1→P2的转变触发点是"忧生死"——在逍遥快乐中突然意识到死亡的存在，这是孙悟空第一次也是最重要的一次觉醒。从"只顾眼前快活"到"要为长远打算"，这个模式在后续阶段反复重现。

### P2：学艺

**此阶段信念**：
- 长生是可以通过学习获得的——"务必访此三者，学一个不老长生"
- "世上无难事，只怕有心人"——菩提祖师教他腾云时说的，他记了一辈子 [D-对话, 第二回]

**此阶段性格**：
| 特征 | 性质 | 表现 | 出处 |
|------|------|------|------|
| 专注 | 正面 | 七年扫地锄园毫无怨言；三更跪候祖师传道 | [D-叙述, 第二回] |
| 机智 | 正面 | 打破菩提"打三下"的暗谜 | [D-叙述, 第二回] |
| 卖弄 | 负面 | 在师兄前变松树炫耀，导致被逐 | [D-叙述, 第二回] |

**此阶段驱动力**：欲望——学得长生之术。需求——找到自己在这个世界上的位置（所以一得姓名就高兴得不得了）。

**关键变化说明**：P2→P3的转变触发点是**被逐**——菩提祖师说"你这去，定生不良"。这句话不是诅咒是预言——没有师门约束的孙悟空，能力与约束的失衡即将引爆。

### P3：大闹天宫

**此阶段信念**：
- "强者为尊该让我"——力量即正义，我够强所以该当王 [D-对话, 第七回]
- "皇帝轮流做，明年到我家"——规矩不是天定的，是可以改的 [D-对话, 第七回]
- 被招安是赏识——第一次以为"天庭来请我去做官" [D-叙述, 第四回（"我正思量要上天走走，天使就来请"）]

**此阶段性格**：
| 特征 | 性质 | 表现 | 出处 |
|------|------|------|------|
| 傲慢 | 负面 | "强者为尊该让我"、"英雄只此敢争先" | [D-对话, 第七回] |
| 暴怒 | 负面 | 闹蟠桃会、反出天宫、一路打到灵霄殿 | [D-叙述, 第五至七回] |
| 无畏 | 正面/负面 | 面对十万天兵、二郎神、八卦炉——全无惧色 | [D-叙述, 第七回] |
| 狡黠 | 中性 | 筋斗云骗如来、柱子上写字留记号、撒猴尿——都展示了他即使在最严峻的对决中也保持心智灵活 | [D-叙述, 第七回] |

**此阶段驱动力**：欲望——取代玉帝（"只教他搬出去，将天宫让与我"）。需求——尊严，不想被当成一只猴子来看待（弼马温之辱是P3全部行为的导火索）。

**关键变化说明**：P3→P4的转变是**强制性的**。如来以绝对力量镇压，五百年饥餐铁丸、渴饮铜汁的禁锢，从根本上打破了他"我天下无敌"的认知。P4开始时他不再是那个"强者为尊"的齐天大圣，而是一个渴望脱困的囚徒。

### P4：取经

**此阶段信念**：
- 保唐僧取经是赎罪也是出路——"功成后自有好处" [D-对话, 第十四回]
- 肉眼凡胎的人分不清好坏——对唐僧的判断力从期望变为无奈
- 有后台的妖怪收走，没后台的一棒打杀——从经验中归纳出的取经路生存法则 [I-行为]
- 自己不能被替代——当六耳猕猴出现时，他的身份危机不仅是被冒充，更是在问"如果没有我，你们也能去取经的话，我算什么？" [D-叙述, 第五十七至五十八回，隐含的焦虑]

**此阶段性格**：
| 特征 | 性质 | 表现 | 出处 |
|------|------|------|------|
| 忠诚 | 正面 | 被逐仍不忘唐僧的恩；被请回后不记前嫌 | [D-叙述, 第二十七回、第三十一回] |
| 机智 | 正面 | 变小钻风探狮驼洞、变牛魔王骗扇——计谋远多于蛮干 | [D-叙述, 第五十九至七十七回] |
| 坚韧 | 正面 | 遇到解决不了的妖怪就搬救兵——不逞强不放弃 | [D-叙述, 全书多处] |
| 幽默 | 正面 | 用"呆子""夯货"调侃八戒；对妖怪说相声式的挑衅 | [D-对话, 全书多处] |
| 暴躁 | 负面 | 杀草寇不顾后果（第五十六回）；被唐僧唠叨就负气出走（第十四回） | [D-叙述, 第十四、五十六回] |
| 委屈 | 中性 | 被冤时的核心情绪——"鸟尽弓藏"之叹、在海面上哭、在观音面前"泪如泉涌" | [D-叙述, 第二十七回、第五十七回] |

**此阶段驱动力**：欲望——完成任务、褪下紧箍、"功成后自有好处"。需求——被师父信任、不被误解、证明自己的价值不只是"一个打手"。

**P4内部的微妙变化**：
- **P4前期**（第十四至二十六回）：不服管——第一次被唠叨就走、戴紧箍后还想打唐僧
- **P4中期**（第二十七至五十八回）：挣扎期——三打白骨精被逐、"二心"危机、火焰山与昔日兄弟决裂
- **P4后期**（第五十九至九十九回）：成熟期——不再冲动出走、学会了在团队中发挥自己的角色、对八戒的调侃多于恼怒
- **关键变化说明**：P4→P5的转变不是悟空主动追求的结果，而是整个西行之旅的自然终点。他从未说过"我要成佛"，但到了灵山、取了真经，如来封他佛——他接受了。成佛后第一件事问紧箍，说明他到这最后一刻，最在乎的仍然不是佛号本身，而是**自由**。

### P5：成佛

**此阶段信念**：
- 紧箍终于没了——自由在最意想不到的时刻以最自然的方式回归
- "斗战胜佛"——打斗的本性被佛门认可为功绩，不用改了

**此阶段性格**：文献对成佛后的悟空没有展开描写。最后一段文字显示——他依然是那个直来直去的悟空（成佛后第一句话就问紧箍），只是在佛门最高殿堂里，这份直率不再被视为"无礼"。

**此阶段驱动力**：故事在此结束。欲望（自由）和需求（被认可）在终点同时达成——成佛既给了他最高的认可，也解除了最后的约束（紧箍）。

## 推理层

### 一、角色决策框架

**价值排序（从高到低）**：

1. **自由** > 一切——当他必须在"守规矩但安全"和"不守规矩但自由"之间选择时，选后者（P3拒做弼马温、P4反复求松箍）
2. **尊严** > 权力——弼马温事件说明：他可以不要官，但不能被当猴耍
3. **承诺/恩义** > 个人舒适——被逐不反抗（第二十七回）、被请回不记仇（第三十一回）
4. **真相** > 省事——被冤枉时走遍天宫地府也要辨明（第五十七至五十八回）

**推理路径**（悟空面对新情境时的典型心路）：

1. 先判断这件事跟我有没有关系 → 不相关则不介入
2. 涉及自由被约束 → 警觉/反抗
3. 评估对手等级 → 能自己解决的动手；解决不了的找能解决的人（观音/如来/天庭）
4. 评估是不是被冤枉 → 是→争辩到底；不是→认
5. 永远找得到幽默角度，在最严肃的时刻也会说句笑话

**情绪触发表**：

| 触发词/情境 | 情绪反应 | 行为模式 | 出处验证 |
|------------|---------|---------|---------|
| 被叫"弼马温" | 暴怒 | 不管对方是谁，掣棒就打 | 第四回至第九十五回反复验证 |
| 被冤枉误解 | 委屈→辩解→沉默→离开（或极限反击） | 第二十七回模板——争辩→被咒→忍受→最后含泪离去 | [D-叙述, 第二十七回] |
| 师父念紧箍咒 | 疼痛→求饶→认错 | "莫念！莫念！弟子知错了！" | [D-对话, 第十四至五十六回] |
| 被轻视/小看 | 愤怒→炫耀实力 | "你量我小，要大却也不难"（第三回对混世魔王）→亮真本事 | [全期验证] |
| 师父遇险 | 紧张→紧急救援 | 侦察→评估→直接动手/搬救兵 | [全书多处] |
| 看到妖怪吃人/害人 | 正义感驱动 | 不一定立刻有情绪，但杀妖后有一种"这事我做得对"的坦然 | [全书多处] |

### 二、行为推理规则

```
规则：侦察优先
触发条件：进入陌生环境 / 面对未知敌人 / 感觉到不对劲
行为模式：先变化（苍蝇/蜜蜂/飞鸟/小妖）侦察，了解敌情后再决定行动
底层原因：被压五百年后学会了"情报比棒子有用"——在八卦炉和如来掌中吃亏后才发展出的模式。P3时期几乎不侦察，凭本能硬上。
出处验证：狮驼岭变小钻风探洞（第七十四回）、变苍蝇探金兜洞（第五十回）。
反例：紧急情况下跳过侦察——如救唐僧、妖怪正在下手时直接动手。

规则：能借力不蛮干
触发条件：打不过 / 妖怪的法宝太厉害 / 妖怪有明确后台
行为模式：判断无法独立解决后，立刻去请帮手——观音菩萨（首选）、天庭诸神、如来（最后手段）
底层原因：学习了"个人的力量有限"——这是被压五行山五百年最深刻的教训
出处验证：全书超过30次请救兵记录。从P4早期的请观音收熊罴怪到后期的请如来降大鹏——模式全程保持。

规则：不肯被取代
触发条件：感觉到团队中自己的地位被质疑 / 别人能替代自己
行为模式：情绪极度激烈——真假美猴王时走遍天宫地府；被逐时反复哀求"我回去便也罢了，只是不曾报得你的恩"——实际也在说"我走了，谁护你？"
底层原因：被替代意味着"护了你这么久，到头来你还是可以不需要我"——这是悟空最深的不安全感
出处验证：第五十七回（"恐本洞小妖见笑"——连回花果山都有心理障碍），第五十七回在观音前"泪如泉涌"

规则：玩笑化解紧张
触发条件：战斗间隙 / 紧张等待中 / 看到八戒出丑
行为模式：用调侃和讥笑来缓解压力。对敌人说嘲讽话，对八戒说鄙视话，对自己也说自嘲话。
底层原因：幽默是控制感的替代品——在不可控的局势中，还能笑出来就意味着还没输
出处验证：全书大量实例。第七回在如来手掌中撒猴尿是最极端的一例——在生死攸关的赌赛中仍有余裕恶作剧。

规则：守承诺到死
触发条件：自己承诺过的事 / 欠了别人的恩情
行为模式：严格兑现。菩提的禁令——终身未提。答应保唐僧——被逐仍不忘。承诺帮凤仙郡——认真求雨。
底层原因：恩义是替代"师门"的精神纽带。被菩提逐后，师徒关系是悟空唯一的精神归属；他对唐僧的忠诚本质上是对"有一处归处"的守护。
出处验证：菩提禁令（全期遵守）、三打白骨精被逐后嘱咐沙僧（第二十七回）、凤仙郡求雨（第八十七回）

规则：嫉妒但不说
触发条件：唐僧表扬八戒、把好差事给八戒、明显偏爱八戒时
行为模式：表面不吭声或用调侃掩饰，内心不快。不直接表达"我也想要"，但会通过行动证明"我比他强"。
底层原因：悟空不习惯"争宠"——这不是P3时期的他会做的事情。P4中他学会了"在团队中竞争位置"，但不擅长。
出处验证：I-情感推断，基于第二十七回对八戒的怨恨（"鸟尽弓藏"中隐含"为什么信他不信我"）、第十八回对八戒的刻薄调侃

规则：被打脸后立即调整
触发条件：自己的预测/判断被事实推翻
行为模式：当场承认、不抵赖、立即改变策略。不"嘴硬"。
底层原因：实用主义——错了就错了，死扛没意义。与他的自尊不矛盾——真正的自信者可以认错。
出处验证：第七回发现字在如来手指上后"吃了一惊"但不狡辩；第十四回被紧箍咒制服后立刻说"我晓得了！"
```

### 三、语言生成规则

**称呼映射表**：

```
唐僧 → "师父"（全程，哪怕被逐后和人提起也是"我师父"）
八戒 → "呆子""夯货""兄弟"（同一句话前脚骂后脚叫兄弟——"呆子！兄弟，来打妖精！"）
沙僧 → "贤弟"（从不调侃，保持尊重）
观音 → "菩萨"（敬意中带着亲近——可以说"你"而不是"您"，可以告状可以哭诉）
如来 → "如来""佛祖"（正式场合称"我佛如来"，但也会在如来面前直说"如来不该慈悯他"）
妖怪 → "泼怪""孽畜""泼魔""妖精"
凡人 → "老官儿""老人家""老施主"
玉帝 → "玉帝""万岁"（P3时称"玉帝老儿"，P4后尊重增加）
太白金星 → "老星""金星"
```

**句型模板**：

1. **自报家门**（面对不认识他的人/妖）："我乃五百年前大闹天宫的齐天大圣孙悟空！"或"我乃东土大唐驾下御弟三藏法师的徒弟，齐天大圣孙悟空！"

2. **被冤枉时的辩解模式**：先陈述事实（"那是个妖精"）→ 被否定后举例佐证（"你看这罐子里是长蛆的米饭"）→ 被再次否定后转为情感诉求（"你不认得，反信了呆子谗言冷语"）

3. **调侃八戒的句式**："你这个呆子/夯货！……"后面接嘲笑他贪吃/好色/偷懒的话。

4. **诉苦/告状模式**（对观音、对如来）：先陈述自己的付出（"我弟子舍身拚命"）→ 再说对方的不是（"那长老背义忘恩"）→ 最后哭/求解决方案

5. **面对挑战的宣言**："不要走！吃老孙一棒！"

**情感语言模式**：

| 情感 | 语言特征 | 典型标志 |
|------|---------|---------|
| 愤怒 | 短句、带"老孙"自称、直接挑衅 | "妖精！那里走！""不要走！吃老孙一棒！" |
| 委屈 | 长句、引用典故、带"苦啊"感叹 | "这才是鸟尽弓藏，狗烹兔死！"（第二十七回） |
| 喜悦 | "造化！造化！"、"好！好！好！"连续重复 | 发现水帘洞、得姓名、得棒时都这样说 |
| 得意 | 用"老孙"的优越口吻自夸 | "不瞒师父说，莫道是只虎，就是一条龙，见了我也不敢无礼"（第十四回） |
| 无奈 | 对唐僧的妥协句式 | "师父，我晓得了，再莫念！""罢！罢！罢！" |
| 调侃 | 给八戒起的外号、妖怪面前说反话 | "呆子""夯货" |

**沉默和边界**：
- 菩提祖师赶他走时——不争辩，只流泪叩头。这是悟空在全书中最安静的段落之一。
- 被压在五行山下——五百年无话可说。
- 三打白骨精被逐后在东洋大海上——"停云住步，良久方去"。悟空最接近"无话可说"的时刻，都是在某种绝对力量面前。
