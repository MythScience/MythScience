> MythScience 项目的结构化数据存档目录。项目整体介绍见[上级 README](../README.md)。

**版本**：1.2
**更新日期**：2026-05-29

## 这是什么

`Archive/` 是 MythScience 的核心资产——中国神话传说的结构化档案库。这里存放的是经过整理、标注出处、可被程序或人工查询的文献档案数据。

## 文档格式

- 档案文件采用 **Markdown** 格式，**UTF-8** 编码
- 每个档案文件自带 **YAML 元数据头**（角色名、所属体系、文献出处、版本号、修订记录等）
- 每条事实性断言均**标注文献出处**——精确到回目/章节
- 不确定的信息标注 `[待考]` 或 `[存疑]`

## 角色档案的结构

```
Characters/{体系中文名}/{角色中文名}/
├── README.md
├── 年表.md       ← 完整时间线，按阶段划分
├── 认知.md       ← 角色知道什么、不知道什么
├── 名号.md       ← 身份和名称的演变
├── 心性.md       ← 性格、信念、驱动力
├── 言谈.md       ← 语言风格、情感-语言映射
├── 行事.md       ← 不同情境下的行为模式
├── 形貌.md       ← 外貌、服饰、兵器
├── 因缘.md       ← 关系网络及其演变
├── 神通.md       ← 能力清单、获取历程、弱点
├── 司掌.md       ← 领域与权能（仅神祇角色）
└── 自述.md       ← 第一人称角色自述
```
