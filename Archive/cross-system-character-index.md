# MythScience 跨体系同名角色索引

**版本**：1.0
**日期**：2026-05-28
**用途**：列出在不同神话体系中出现的同名（或实质同源）角色，提供各体系内独立档案的路径映射。

## 设计原则

1. 本文件不合并角色——各体系内的同名角色是独立实体。不同著作中的同名角色（如《西游记》哪吒与《封神演义》哪吒）分别建档，互不引用
2. 仅提供**索引映射**——告诉用户"这个角色在哪些体系中有档案、分别在哪里"
3. 跨体系对比由应用层处理，不在档案层面实现

## 扩展指引

当新体系或新角色加入时：

1. 在对应节下增加体系行
2. 如果角色尚未出现在任何现有节中，新建一个角色节
3. 状态列使用：已完成 / 进行中 / 规划中
4. "体系间主要差异"仅写已有档案或已知文献中已确认的核心差异，不推测未编写的档案内容
